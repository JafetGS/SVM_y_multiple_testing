# SVM_y_multiple_testing

Se trabajara con la base de datos de Khan, que consiste de 83 muestras y 2308 variables de entrada, que consisten en la expresión génica estandarizada de distintos genes. La variable de salida cuenta con valores numéricos del 1 al 4 que corresponden a distintos tipos de cáncer.

El objetivo de este proyecto es:
  1. Comparar y medir la diferencia de medias de los genes de distintas clases.
  2. Calcular y comparar los genes mas significativos mediante el uso de tres metodologias distintas (Bonferroni, Holm, Benjamini-Hochberg) de dos clases en particulares.
  3. Calcular y comparar los genes mas significativos mediante el uso de ANOVA de todas las clases.
  4. Entrenar tres modelos SVM capaces de predecir la clase a la que pertenece un tipo de cancer (lineal, polinomial de grado 3, radial.
  5. Medir el desempeño de los modelos mediante distintas metricas, y verificar si hay un kernel en particular que sea superior a las demas para esta tarea en especifica.
